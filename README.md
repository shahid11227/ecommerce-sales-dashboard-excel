# ecommerce-sales-dashboard-excel

## 📊 Project Overview
This project is an interactive E-commerce Sales Dashboard built using Microsoft Excel. 
It provides insights into sales, profit, quantity, and order trends using dynamic visualizations and KPIs.

##  Objectives
- Analyze overall sales performance
- Track profit and profit margin
- Identify top-performing categories and sub-categories
- Monitor year-over-year (YOY) growth
- Provide interactive filtering using slicers

##  Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers & Filters
- Data Cleaning

## Dashboard Features
- KPI Metrics (Sales, Profit, Quantity, Orders, Profit Margin)
- YOY Growth Analysis
- Category-wise Profit Analysis
- Sales Distribution (Donut Chart)
- State-wise Sales (Map)
- Top Sub-category Sales
- Interactive Filters (Year, Region, Segment)
  
##  Key Insights
- Sales showed consistent growth over time
- Technology category generated highest profit
- Certain sub-categories contributed major revenue
- Profit margin remained stable across years

##  How to Use
1. Download the Excel file
2. Open in Microsoft Excel
3. Use slicers to filter data by Year, Region, and Segment
4. Explore charts and KPIs for insights
